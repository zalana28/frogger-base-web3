# Base Frogger Web3 Mini Game

A simple Frogger-style game built with React + TypeScript + Canvas on Base.

## Features

- Wagmi + viem wallet connection on Base
- Transaction gate before each play
- Builder Code attribution (`base:app_id` + tx `dataSuffix`)
- Mobile-first controls
- Local leaderboard

## Run

```bash
npm install
npm run dev
```

Open the local Vite URL in your browser.
